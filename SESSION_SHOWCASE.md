# THE LONG LIGHT — School World  
## Grok 4.5 single-session forge showcase

**Date:** 2026-07-09  
**Engine:** Godot 4.3 (not pygame, not HTML)  
**Builder:** Grok 4.5 in Grok Build with Mac (Lycheetah)  

### What this is
A **Lycheetah Mystery School RPG** — Pokémon/DS world feel, D&D stats, framework combat (MEASURE / COMPRESS / TRANSMUTE / BREAK…), full alchemy arc to Rubedo, secrets, trainers, interiors.

### Honest register
This is a **dense vertical product**, not Nintendo production art. Systems, map graph, combat, secrets, and ending exist and run. Art/music are runtime/procedural.

### Play
```bash
# needs Godot 4.3 binary (tools/godot next to project or GODOT= path)
bash launch.sh
```

### Session milestones forged
1. Godot project scaffold + autoloads  
2. Multi-area world + quests + combat kit  
3. GBA-style tileset/walk cycles/UI chrome  
4. Day/night, minimap, encounter flash, ambient SFX  
5. Grid movement, interiors, trainers, location splash  
6. Secrets: false walls, digs, cracks, bushes, wanderers, fireflies  
7. Hidden Grotto + Starwell + secret boss *The Hidden*  
8. Rubedo + *The Unfinished Work* + ending  
9. Name entry, LOS trainer eyes, random world whispers  

### Controls (short)
Grid walk · Shift run · E interact · Esc menu · 1–0 battle skills · F flee
